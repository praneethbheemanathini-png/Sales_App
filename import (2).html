Bulk Import Catalog

Admin-only tool. Upload your product Excel file once to load many products into FieldOrder at the same time. This page is not part of the salesman app — keep it private, or delete it from your repo after you're done importing.
Looks for a sheet named "Catalog" with columns: SERIES NAME, NAME, MRP, Retailer price, COLOR, Is HOT selling, Image. Product name is built as "SERIES NAME + NAME". For images: upload all your image files into a folder named images in the same GitHub repo as this page — the tool will match each product's Image filename to images/<filename> automatically. If a filename isn't found in that folder later, the picture just won't show — nothing breaks.
