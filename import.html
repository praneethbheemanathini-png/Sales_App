Bulk Import Catalog

Admin-only tool. Upload your product Excel file once to load many products into FieldOrder at the same time. This page is not part of the salesman app — keep it private, or delete it from your repo after you're done importing.
Looks for a sheet named "Catalog" with columns: SERIES NAME, NAME, MRP, Retailer price, COLOR, Is HOT selling, Image. Product name is built as "SERIES NAME + NAME". Image column must contain a full web link (https://...) to actually show a photo — plain filenames like "C4Pro.png" won't display unless you host those images somewhere and update the links.
